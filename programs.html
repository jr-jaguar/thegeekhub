Programs

  • Currently, the Geekhub offers two programs: “Front-end development” as well as “Advanced PHP and frameworks”.

    The “Front-end development” course deals with all things that the end user interacts with in the system. Students in this course learn semantic markup, (X)HTML, CSS, Javascript, AJAX, integration with popular services like Facebook, Twitter, Paypal, Authorize.net, etc. With the popularity of Web 2.0 applications, front-end development has received a lot of attention since the concept of usability has become very important. At the end of the “Front-end development” course, students will be able to create complex pages with rich media, animation and user interaction elements.

    The “Advanced PHP and frameworks” course focuses on server side logic. In this class, students are learning object oriented programming and the most popular PHP framework to date — Symfony. Tasks solved by this group include calculations, data management, user accounts, permissions, content management, integration with system and 3rd party libraries, etc. Students of this course are able to implement business logic for web-based products.

    Next year we are planning to add mobile technologies as a course of choice. Students will be learning “Java for Android”.
