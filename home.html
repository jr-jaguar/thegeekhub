The courses


  • Front-end development

    The “Front-end development” course deals with all things that the end user interacts with in the system. Students in this course learn semantic markup, (X)HTML, CSS, Javascript, AJAX, integration with popular services like Facebook, Twitter, Paypal, Authorize.net, etc.


  • Advanced PHP and frameworks

    In this class, students are learning object oriented programming and the most popular PHP framework to date — Symfony. Students of this course are able to implement business logic for web-based products.
